# Realtime Collaborative Editor

A real-time collaborative text editor built using Node.js, Express.js, WebSockets, and CodeMirror.

---

## Features

- Real-time text synchronization
- Multi-user collaboration
- Automatic reconnection handling
- Live collaborative editing
- WebSocket-based communication

---

## Tech Stack

- Node.js
- Express.js
- WebSockets
- JavaScript
- CodeMirror

---

## Installation

```bash
npm install
node server.js
```

Open in browser:

```bash
http://localhost:3000
```

---

## Project Overview

This project is a real-time collaborative editor where multiple users can edit text simultaneously. Any changes made by one user are instantly reflected across all connected clients using WebSocket communication.

The application uses CodeMirror as the editor interface and Node.js with Express.js for backend server functionality.
