# Realtime Collaborative Editor

A real-time collaborative text editor built using Node.js, Express, WebSockets, and CodeMirror.

## Features

- Real-time text synchronization
- Multi-user collaboration
- Automatic reconnection handling
- Live editing using WebSockets
- CodeMirror editor integration

## Tech Stack

- Node.js
- Express.js
- WebSockets
- JavaScript
- CodeMirror

## Installation

```bash
npm install
node server.js
